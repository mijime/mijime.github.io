Title: Google tag manager の概要
Date: '2017-10-22T03:07:56+09:00'
Draft: false
---

Google tag manager は HTML タグを動的に挿入するサービス。

- 既存の HTML に手を加えなくても管理画面から追加できる
- 挿入する HTML タグを一元に管理できる

のが強み。

挿入できる HTML タグのテンプレートが用意されており

- google analytics
- doubleclick

などはすぐに開始できる。

カスタムの HTML を挿入することも可能。
