Google tag manager の概要

Sun, Oct 22, 2017

Google tag managerはHTMLタグを動的に挿入するサービス。

  • 既存のHTMLに手を加えなくても管理画面から追加できる
  • 挿入するHTMLタグを一元に管理できる

のが強み。

挿入できるHTMLタグのテンプレートが用意されており

  • google analytics
  • doubleclick

などはすぐに開始できる。

カスタムのHTMLを挿入することも可能。

Reference

  • https://www.google.com/intl/ja/analytics/tag-manager/
