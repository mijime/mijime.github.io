# mintodo

Browser-based mindmap todo editor (React + TypeScript).

## Commands

```bash
pnpm test         # run tests
pnpm run check    # type check + lint
pnpm run format   # format
pnpm run dev      # vite dev server
```

## Stack

- Runtime/package manager: **pnpm**
- Test runner: `vitest`
- Type check: `tsgo --noEmit`
- Lint: `oxlint`
- Format: `oxfmt`
- Build: `vite build`
