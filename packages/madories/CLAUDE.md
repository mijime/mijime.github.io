# madories

Browser-based floor plan editor (React + TypeScript).

## Commands

```bash
pnpm test         # run tests
pnpm run check    # type check + lint
pnpm run format   # format
```

## Stack

- Runtime/package manager: **pnpm**
- Test runner: `vitest`
- Type check: `tsgo --noEmit`
- Lint: `oxlint`
- Format: `oxfmt`
